# **App Name**: Innlink Data

## Core Features:

- Data Bundle Purchase: Integrate APIs for MTN, Telecel, and AirtelTigo to allow users to purchase data bundles.
- Result Checker Purchase: Enable one-tap purchase of BECE, WASSCE, and CSSPS Placement cards, delivering the PIN via SMS, Email, and In-App display.
- Airtime and Bills Payment: Support airtime top-ups across all networks and utility payments (ECG, Ghana Water).
- Queue-Based Transaction Engine: Implement a queue-based system to handle network API downtime, notifying users and queuing requests for immediate fulfillment once the network is back online.
- Dual-Wallet System: Allow users to pay directly via MoMo or maintain an in-app Prepaid Wallet for faster checkout.
- Admin Dashboard: Create a control center to manually adjust prices, view real-time network success rates, and manage Result Checker stock.
- Network Auto-Detection: Automatically detect the network provider as soon as a user types the first three digits of a phone number.

## Style Guidelines:

- Primary color: Deep blue (#2c3e50) for a professional and trustworthy feel.
- Background color: Off-white (#f0f4f8) to create a clean and modern base.
- Accent color: Golden yellow (#ffc107), inspired by Ghanaian culture, to highlight key actions and information, complemented by network-specific colors like MTN Yellow, Telecel Red, and AirtelTigo Blue/Red.
- Font pairing: 'Poppins' (sans-serif) for headlines to provide a contemporary and precise feel, combined with 'PT Sans' (sans-serif) for body text to ensure readability and a modern look.
- Use clear, modern icons with subtle animations for data provisioning success. Design success haptics for user feedback.
- Employ a super-app layout (similar to Hubtel or PalmPay) for a comprehensive and user-friendly experience.
- Implement animated progress bars and success haptics to provide real-time feedback on transactions.
